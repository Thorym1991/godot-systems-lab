# Godot Systems Lab — Project Overview

This project is a modular Godot 4 framework designed to quickly build gameplay systems.
All features are structured as reusable components, plus demo scenes that show how to use them.

---

## Current Systems

| System | Status | Description |
|---|---|---|
| Player State System | ✔ Completed (base) | Shared core logic for movement states |
| Platformer Player | ✔ Demo ready | Idle / Move / Air, jump + gravity |
| Topdown Player | ✔ Demo ready | 4-way / 8-way movement logic |
| DemoHub | ✔ Implemented | Launch scenes and return with HUD button |
